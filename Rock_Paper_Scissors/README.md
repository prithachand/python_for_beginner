# Rock Paper Scissors

A beginner-friendly Python game where you play Rock Paper Scissors against the computer.

## Features

- Choose `rock`, `paper`, or `scissors` each round.
- The computer randomly chooses its move.
- The game announces the winner of each round.
- Score tracking keeps count of player wins, computer wins, and ties.
- Type `q` to quit and see the final score.

## How to Run

From the repository root, run:

```bash
python3 Rock_Paper_Scissors/rock_paper_scissors.py
```
